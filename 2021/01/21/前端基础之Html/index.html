html是什么

  1. 超文本标记语言
    html用文本来描述网页，但是不是普通的文本，超出文本之外
    标记语言： 标记标签语言，是一套标签，也就是说用标签来描述网页

  2. html不是编程语言。是由浏览器直接解释执行

标签有很多。标签是成对出现，也就是说有开始，有结束，也有部分自闭合标签（没有结束标签)

编写html

  1. 文本编写

    1. 新建文件hello.html

    2. 编写
      语法结构：

       <html>
 <head>
     <!-- html属性，js，css等设置区域 -->
 </head>
 <body>
     浏览器的可见区域
 </body>
 </html>

    3. 运行
      使用浏览器打开文件

  1. 使用集成工具编写

    1. 新建web工程
      new –> web Project

    2. 在pages下新建页面
      new –> others—>html

    3. 使用浏览器打开文件

15. 超链接

    <a>,href属性和target属性
    <a href="" target="">xxx可用图片</a>

href : 跳转页面

16.锚链接

在当前页面做快速定位

  1. 定义锚点

         <a name="锚点">xxx</a>

  2. 定义锚链接，href指向锚点

         <a href="#锚点">xxx</a>

target : 指定页面跳转方式；如_blank新窗口打开
href : 跳转页面
